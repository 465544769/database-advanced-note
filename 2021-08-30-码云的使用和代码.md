## 全局配置
```  
git config --global user.name "<用户名>"  
git config --global user.email "<邮箱>"
```
## 下载库到本地
```
git clone <库地址>
```

#### 提交到本地仓库
```
git add .  
git commit -m "<备注>"
```

#### 从本地仓库上传到云端仓库
```
git push
```

#### 从云端仓库更新到本地仓库
```
git pull
```

#### 查询历史版本  
```
git log
```

#### 返回某个版本  
```
git reset --hard <版本哈希值>
```
